# DevTrack Pro

Full Stack Task & Time Tracking System

## 🚀 Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Auth: JWT

---

## 🛠 Setup

### 1. Clone repository

```bash
git clone https://github.com/yourname/devtrack-pro.git
cd devtrack-pro
```

### 2. Setup Backend

```bash
cd server
cp .env.example .env
npm install
npm run dev
```

### 3. Setup Frontend

```bash
cd client
npm install
npm start
```

Frontend runs on:
http://localhost:3000

Backend runs on:
http://localhost:5000

---

## ✨ Features

- User registration & login
- JWT authentication
- Create and fetch tasks
- Protected API routes
- MongoDB persistence
